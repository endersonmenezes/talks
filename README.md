# Public Talks & Teaching Hub

Bem-vindo ao repositório público centralizado para minhas palestras, participações em eventos, cursos lecionados e workshops práticos. Este espaço é mantido sob a filosofia de [Building in Public](https://buildinpublic.com/manifesto) e do [Open Source](https://opensource.org/about).

Também mantenho meu perfil no [Sessionize](https://sessionize.com/endersonmenezes/) para futuras submissões de palestras.

## 📂 Estrutura do Diretório

Esta pasta está dividida nos seguintes temas:

- **[`events/`](events/)**: Registros e materiais de eventos que participei ou palestrei.
- **[`presentations/`](presentations/)**: Tópicos e apresentações base que costumo ministrar.
- **[`courses/`](courses/)**: Materiais didáticos completos de cursos e disciplinas universitárias ou de pós-graduação.
- **[`workshops/`](workshops/)**: Trilhas práticas e laboratórios estilo *hands-on*.

## Convenções de Contribuição Interna (Vault)
- **Pastas**: Segregação conforme listado acima.
- **Date**: YYYY-MM-DD
- **Filenames**: Eventos como `YYYY-MM-DD_event-slug.md`, palestras base como `talk-slug.md`.
- **Imagens**: Devem ir preferencialmente para o nível `assets/YYYY-MM-DD_event_photo1.jpeg`.
- **Frontmatter YAML**: Padrão do cofre (title, type, status, area, tags, created, updated, source, etc).
